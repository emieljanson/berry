# Berry Icons

Place PNG icons here with transparent backgrounds.

## Required Icons

| Filename | Description | Recommended Size |
|----------|-------------|------------------|
| `play.png` | Play button | 60x60 |
| `pause.png` | Pause button | 60x60 |
| `prev.png` | Previous track | 48x48 |
| `next.png` | Next track | 48x48 |
| `volume.png` | Volume/speaker | 48x48 |

## Tips

- Export as PNG with **transparent background**
- Use **white** (#FFFFFF) as the icon color
- Export at 2x size for crisp rendering (e.g., 120x120 for play, then scale down)
- Icons should be centered within their canvas
